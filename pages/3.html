What's missing?

There are several features that would be nice to have but that haven't made it yet:

  • Showing the title of the next and previous step.
  • Being able to drill down into the errors from both the REPL and the loaded file
  • Highlighting the line that caused an error.
  • Being able to change the theme of the editor.
  • Being able to save your session for a later date.
  • Being able to jump directly to a step in the tutorial.
  • Having multiple files opened at once (perhaps in tabs?)
  • Seeing the current namespace in the REPL.
