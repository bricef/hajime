The editor and REPL

On the right are the editor pane at the top and the REPL at the bottom. The REPL environment is automatically updated with the editor code.

For example, try running (hello "bob") in the REPL.

The REPL is a fully featured Clojure REPL, with all features enabled. Try defining a goodbye function via the REPL.

To help learners it is also possible to add demo code into the editor directly from the tutorial. For example, try clicking on the add to session link beneath the following code.

(defn square [x] (* x x))

You should now be immediately able to use square in the REPL.
